Experience:

Undergraduate Researcher (05/2023 - 01/2024)

I contributed to a research project with Dr. Tang of JMU's Geography department focused on near real-time land change monitoring using the Microsoft Planetary Computer platform. By leveraging Python and LandSat satellite imagery, I helped design and implement an algorithm capable of detecting and quantifying environmental changes over time. The system integrated high-resolution Microsoft imagery to train the model and generate a robust land change score, allowing for accurate comparison between observed and predicted landscape alterations.

To support this work, I applied advanced data preprocessing and modeling techniques using tools such as pandas, scikit-learn, and SciPy. These tools enabled the efficient manipulation of large satellite datasets and the development of reliable predictive models. Additionally, I created detailed visualizations with matplotlib to clearly communicate spatial and temporal trends in land change, providing valuable insights into environmental dynamics.
