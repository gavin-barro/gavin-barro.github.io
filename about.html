Additionally, I spent 6 months collaborating with Dr. Tang from the ISAT/Geography department on a research project focused on near real-time environmental monitoring. Our work involved implementing a sophisticated algorithm on the Microsoft Planetary Computer platform, leveraging its vast repository of LandSat data. Using Python, we developed a methodology to train LandSat imagery with auxiliary data from Microsoft, generating a unique land change score. This score allowed us to quantitatively compare observed land changes against predicted values derived from LandSat, with our findings meticulously analyzed and visualized using key Python libraries such as pandas, scikit-learn (sklearn), SciPy, and Matplotlib.

AWS Certified Cloud Practitioner

Earning the AWS Certified Cloud Practitioner certification has equipped me with a foundational understanding of cloud computing and Amazon Web Services (AWS), empowering me to navigate and contribute to cloud-based projects with confidence. This entry-level credential validates my ability to grasp core AWS services, cloud concepts, and best practices, making it a valuable asset for both technical and non-technical roles. Through dedicated study, I mastered the essentials of cloud technology, preparing me to collaborate effectively with technical teams and support cloud adoption initiatives.

Skills Gained:

  • Cloud Fundamentals: Developed a solid understanding of cloud computing concepts, including deployment models like IaaS, PaaS and SaaS, and their applications in real-world business scenarios.
  • AWS Services Knowledge: Gained familiarity with key AWS services such as EC2 (compute), S3 (storage), and basic networking and database solutions, enabling me to identify and propose cloud-based solutions.
  • Global Infrastructure: Learned to leverage AWS's global infrastructure, including Regions, Availability Zones, and Edge Locations, to optimize resource management and performance.
  • Cost Management: Mastered AWS pricing models and cost optimization strategies, using tools like the AWS Pricing Calculator to make informed, cost-effective decisions.
  • Security Basics: Acquired knowledge of the AWS Shared Responsibility Model and fundamental security services, enhancing my ability to contribute to secure cloud operations.
  • Well-Architected Framework: Understood the principles of the AWS Well-Architected Framework, allowing me to support the design of secure, efficient, and reliable cloud architectures.

This certification has strengthened my ability to communicate cloud concepts clearly, bridge the gap between technical and business teams, and lay a strong foundation for further growth in cloud computing.
